# Changelog

All notable changes to Astra UI are documented here.

## 1.1.0 - 2026-06-12

### Added

- Rebranded the interface as Astra UI with a violet deep-space default theme.
- Added Astra, Nebula, Eclipse, SolarFlare, Aurora, and Starlight presets.
- Added distinct code-rendered marks and themed badges for tab navigation.
- Added `UI:SetAccent()`, `UI:GetThemes()`, and color-picker `ApplyAccent`.
- Added an Astra title-bar mark and live state repainting after theme changes.
- Added `dist/AstraUI.lua` while retaining the old bundle name for compatibility.

## 1.0.0 - 2026-06-12

### Added

- Modular Luau source architecture and deterministic executor bundle.
- Draggable, resizable, minimizable multi-window system.
- Collapsible sidebar, tabs, search, lazy loading, and collapsible sections.
- Buttons, toggles, sliders, dropdowns, multi-dropdowns, inputs, keybinds,
  color pickers, paragraphs, dividers, images, notifications, and dialogs.
- Theme, config, state, event, flag, animation, and cleanup managers.
- Eleven built-in themes and custom theme registration.
- JSON config import/export with executor file persistence when available.
- Mobile and PC layouts, responsive scaling, tooltips, dependencies,
  watermark, FPS counter, localization, acrylic blur, and executor detection.
